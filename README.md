# Adwaidh CR — Portfolio

  ## Prerequisites
  - Node.js 18+ (https://nodejs.org)
  - npm (comes with Node.js)

  ## Run locally

  ```bash
  npm install
  npm run dev
  ```

  Then open http://localhost:5173 in your browser.

  ## Build for production

  ```bash
  npm run build
  npm run preview
  ```
